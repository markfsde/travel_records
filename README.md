# 旅迹 · 我的旅行手账地球 🌍

一个手绘水彩风格的 3D 旅行手账。在可旋转的地球上点亮去过的地方，给每个地点贴上拍立得照片、到访日期和手写心情，像翻开一本旅行日记。纯前端、无后端、一个 HTML 文件搞定。

## 功能

- 水彩质感的 3D 地球，自动缓慢旋转，可拖拽 / 缩放 / 双指捏合
- 去过的地方亮起墨水光点 + 脉动光环，鼠标悬停显示地名
- 点击光点翻开手账卡片：拍立得照片轮播、日期邮戳、手写笔记
- 「记一笔」表单：输入地名一键定位经纬度，上传本地照片（自动压缩）
- 「飞到这里」让地球转到对应位置
- 数据存浏览器本地（localStorage），支持导出 / 导入 JSON 备份，方便跨设备迁移

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库，把 `index.html` 上传上去（可连同本 README）
2. 进入仓库 **Settings → Pages**
3. Source 选 **Deploy from a branch**，Branch 选 `main` / 根目录 `/ (root)`，保存
4. 等一两分钟，访问 `https://你的用户名.github.io/仓库名/` 即可

> 不需要任何构建步骤，上传即用。

## 说明

- **照片存在浏览器本地**，换电脑 / 清缓存会丢失，记得用「导出备份」存一份 JSON。
- 定位功能优先用内置的常见城市表（离线可用），找不到时才联网查询（OpenStreetMap）。GitHub Pages 是 https，联网定位正常工作。
- 照片上传时自动压缩到 1000px / JPEG，避免占满本地存储；存不下时会提示。

## 技术栈

Three.js (r128) + 原生 JS / CSS，零依赖打包。地球贴图是用 Canvas 程序化绘制的水彩质感，不依赖外部图片。
